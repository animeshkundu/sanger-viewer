2026-07-04 — keyboard-accessible base inspector (FU-1)

July 4, 2026

This pass closes QA follow-up FU-1 by giving keyboard users a first-class way to inspect per-base values. Focusing a base call in the sequence panel now opens an inspector popover with exact position, called base, PHRED score, and peak amplitude.

The inspector opens on focus and keyboard activation (Enter/Space), closes on Escape or when focus leaves the sequence panel, and exposes explicit dialog semantics for assistive tech. Existing hover tooltip behavior is preserved for pointer users.

Validation was added with exact-value Playwright assertions (fixture-backed expected position/base/quality/amplitude), close behavior checks, ARIA wiring checks, and light + dark mode coverage.
