2026-07-04 — plasmid/linear map + restriction overlay linked to trace evidence

I shipped the Top-1 post-v1.0 slice: a sequence map that can flip between circular plasmid and linear views, renders ORF + restriction features from the active read, and stays synced to the current chromatogram window.

Every map marker is keyboard-operable and focus-visible. Clicking (or pressing Enter on) a site/feature now jumps to the exact base range in the trace and opens the base inspector at the exact start base, so the map and trace evidence stay locked together.

Validation is exact-value, not pixel-noise: map tests assert known HindIII coordinates, exact computed marker angles/ratios in circular vs linear modes, and exact jump target base indices. Circular-origin split handling is covered with explicit wrap-range unit tests.

The privacy wedge remains intact: all rendering, restriction scanning, and navigation run fully client-side in browser memory; trace data is never uploaded.
